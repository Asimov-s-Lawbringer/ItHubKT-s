## Это система онлайн школьного образования

### Фишки:

-ООП (Инкапсуляция данных,Полиморфизм методов,Наследование классов)

-Курс Менеджмент без Баз Данных!

-Система отправки Домашних заданий и их проверки учителем с баллами!

-Полноценные роли пользователей: Администраторы, Учителя и Ученики

### Для запуска:

ts-node app.ts
